Class Matrix2D

Custom Matrix2D class used mainly to not depend on DOM in test and unrelated to rendering systems. Also DOMMatrix is unstable. Represents a 3x3 matrix for 2d transformations, uses @class Vector2 Indexes are the following: i - row, j - column, starting from 0 to 2 inclusive.
